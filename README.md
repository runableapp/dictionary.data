# dictionary-data

This project publishes SQLite dictionary databases packaged as `.zip` files, mainly for Korean-related dictionary data.

These files are prepared for **[Runable.app's dictionary software](https://github.com/runableapp/dictionary)**.

ZIP assets are available in this repository and on the GitHub Releases page:

- https://github.com/runableapp/dictionary.data/releases

## ZIP assets

### Combined package

- `dicts/dictionary_fts5.zip` - Combined FTS5-ready dictionary database package used by the app.

### Individual dictionary packages (`dicts/*.zip`)

| ZIP file | Dictionary | Language direction |
|---|---|---|
| `dicts/cc-kedict.zip` | CC Korean-English Dictionary | 🇰🇷 Korean -> English 🇺🇸 |
| `dicts/kaikki-korean.zip` | Kaikki Korean lexical data | 🇰🇷 Korean -> Korean 🇰🇷 |
| `dicts/kengdic.zip` | KEngDic Korean-English dictionary | 🇰🇷 Korean -> English 🇺🇸 |
| `dicts/lingoes-concise-e-k.zip` | Lingoes concise English-Korean dictionary | 🇺🇸 English -> Korean 🇰🇷 |
| `dicts/lingoes-concise-k-e.zip` | Lingoes concise Korean-English dictionary | 🇰🇷 Korean -> English 🇺🇸 |
| `dicts/lingoes-indonesian-korean.zip` | Lingoes Indonesian-Korean dictionary | 🇮🇩 Indonesian -> Korean 🇰🇷 |
| `dicts/lingoes-korean-indonesian.zip` | Lingoes Korean-Indonesian dictionary | 🇰🇷 Korean -> Indonesian 🇮🇩 |
| `dicts/lingoes-naver-chinese-korean.zip` | Naver Chinese-Korean dictionary via Lingoes | 🇨🇳 Chinese -> 🇰🇷 Korean |
| `dicts/lingoes-naver-english-korean.zip` | Naver English-Korean dictionary via Lingoes | 🇺🇸 English -> Korean 🇰🇷 |
| `dicts/lingoes-naver-japanese-korean.zip` | Naver Japanese-Korean dictionary via Lingoes | 🇯🇵 Japanese -> Korean 🇰🇷 |
| `dicts/lingoes-naver-korean-chinese.zip` | Naver Korean-Chinese dictionary via Lingoes | 🇰🇷 Korean -> 🇨🇳 Chinese |
| `dicts/lingoes-naver-korean-english.zip` | Naver Korean-English dictionary via Lingoes | 🇰🇷 Korean -> English 🇺🇸 |
| `dicts/lingoes-naver-korean-japanese.zip` | Naver Korean-Japanese dictionary via Lingoes | 🇰🇷 Korean -> Japanese 🇯🇵 |
| `dicts/stardict-hanja-koreanhanzi.zip` | Stardict Hanja (漢字)-Korean dictionary | 🇨🇳 Hanja (漢字) <-> 🇰🇷 Korean |
| `dicts/stardict-koreananimalmedical.zip` | Stardict Korean animal/medical terminology dictionary | 🇰🇷 Korean -> Korean 🇰🇷 (specialized) |
| `dicts/stardict-koreandic.zip` | Stardict general Korean dictionary | 🇰🇷 Korean -> Korean 🇰🇷 |
| `dicts/stardict-koreanenglish.zip` | Stardict Korean-English dictionary | 🇰🇷 Korean -> English 🇺🇸 |
| `dicts/stardict-koreanlaw.zip` | Stardict Korean law terminology dictionary | 🇰🇷 Korean -> Korean 🇰🇷 (legal) |
| `dicts/stardict-koreanmedical.zip` | Stardict Korean medical terminology dictionary | 🇰🇷 Korean -> Korean 🇰🇷 (medical) |
| `dicts/stardict-quick_eng-kor.zip` | Stardict quick English-Korean dictionary | 🇺🇸 English -> Korean 🇰🇷 |
| `dicts/stardict-quick_kor-eng.zip` | Stardict quick Korean-English dictionary | 🇰🇷 Korean -> English 🇺🇸 |
| `dicts/wiktionary-dict-en-ko.zip` | Wiktionary English-Korean dictionary data | 🇺🇸 English -> Korean 🇰🇷 |
| `dicts/yomichan-krdict-kr-jp.zip` | KRDict Korean-Japanese dictionary in Yomichan format | 🇰🇷 Korean -> Japanese 🇯🇵 |
| `dicts/yomichan-naver-kr-jp.zip` | Naver Korean-Japanese dictionary in Yomichan format | 🇰🇷 Korean -> Japanese 🇯🇵 |

## Dictionary source sites

- StarDict: https://stardict.uber.space/
- Lingoes: https://www.lingoes.net/en/dictionary/
- Kaikki: https://kaikki.org/dictionary/
- Wiktionary: https://github.com/open-dsl-dict/wiktionary-dict
- KRDict (National Institute of Korean Language): https://krdict.korean.go.kr/eng
- Yomichan Korean dictionaries: https://github.com/Lyroxide/yomichan-ko-dic
- CC Korean-English Dictionary: https://github.com/mhagiwara/cc-kedict
- KEngDic: https://github.com/garfieldnate/kengdic

## 🇰🇷 한국어

# dictionary-data

이 프로젝트는 SQLite 사전 데이터베이스를 `.zip` 형식으로 배포하며, 주로 한국어 관련 사전 데이터를 대상으로 합니다.

이 파일들은 **[Runable.app의 사전 소프트웨어](https://github.com/runableapp/dictionary)**를 위해 준비되었습니다.

ZIP 에셋은 이 저장소와 GitHub Releases 페이지에서 받을 수 있습니다:

- https://github.com/runableapp/dictionary.data/releases

## ZIP 에셋

### 통합 패키지

- `dicts/dictionary_fts5.zip` - 앱에서 사용하는 통합 FTS5 사전 데이터베이스 패키지.
